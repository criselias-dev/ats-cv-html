# ATS-Friendly HTML CV

Professional CV developed in pure HTML and CSS, focused on ATS compatibility, clean structure, and single-page PDF export.

## Project Purpose
- Demonstrate semantic HTML5
- Apply ATS-friendly layout and typography
- Control print and PDF formatting
- Practice Git and GitHub workflow

## Files
- index.html – main version
- CrisEliasCV.html – visual version
- CrisEliasCVATSfriendly.html – ATS-optimized version
- profile.jpg – profile image

## How to View
Open `index.html` in any modern browser.
